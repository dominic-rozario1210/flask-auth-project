# Flask Authentication Project

This is a simple Flask authentication project built using:

- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript

## Features

- User Registration
- Login System
- Password Hashing
- Session Management
- Logout Functionality

## Technologies Used

- Flask
- mysql.connector
- Werkzeug Security
- Fetch API

## How to Run

1. Install Python
2. Install Flask
3. Install MySQL Connector

```bash
pip install flask
pip install mysql-connector-python
```

4. Run app.py

```bash
python app.py
```
